# yup-perl

```
NAME:
   perl - perl command wrapper for yupsh

USAGE:
   perl [OPTIONS] [FILE...]

      Execute Perl scripts.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --execute value, -e value  execute script
   --module value, -M value   use module
   --include value, -I value  directory for library modules
   --encoding value           specify encoding
   --in-place, -i             edit files in place (default: false)
   --print, -p                assume loop like -n but print line also (default: false)
   --loop, -n                 assume loop around script (default: false)
   --autosplit, -a            autosplit mode with -n or -p (default: false)
   --check, -c                check syntax only (runs BEGIN and END blocks) (default: false)
   --warnings, -w             enable warnings (default: false)
   --strict                   enable strict mode (default: false)
   --debug, -d                run program under debugger (default: false)
   --taint, -T                enable taint checks (default: false)
   --help, -h                 show help
```
